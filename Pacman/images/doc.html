O Jogo Pac Man


Pac-Man é um jogo eletrônico criado pelo Tōru Iwatani para a empresa Namco, e sendo distribuído para o mercado americano pela Midway Games.


O professor Tarcísio Nunes propos uma atividade para os alunos do AIPR 2019 onde deveriam escolher um jogo classico e recrialo no p5js.

Nosso grupo escolheu recriar o jogo do Pac-Man.

Paredes


Este é o elemento mais complexo do jogo!.



O código das Paredes.

Cada - representa uma Moeda!.

Cada * representa uma parede (bloco).

As letras R, G, I e U representam os fantasma.

As duas letras D representam a saida do Fantasma do Centro.

As / representam os espaços em branco (onde não possuem objetos).

As quatro letra O representam as frutas que dão poder.

As letra P representa o Spam do Pac Man.
